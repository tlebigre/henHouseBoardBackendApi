Init my GPIO with :
- 17 --> Button for up
- 27 --> Button for down
- 13 --> Engine run
- 19 --> Engine direction
- 26 --> Engine stop

My hardware :
- Engine reference : 23HS22-2804S
- Step driver reference : DM556 (5.6A (2.8*2), Half Current, 20000 pulse/rev)
- RTC module : HW-084
